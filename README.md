# MyBlogSite

A simple Blog Website built with **Flask**, **SQLite3**, and **Python**.  
This project allows users to:

- Register and Login 
- Create, Edit, and Delete blog posts
- Add comments and ratings
- Browse all blogs by category

## Tech Stack ⚙️
- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite3
- HTML / CSS / Bootstrap
